# health-tracker

个人血脂与用药安全追踪网站。

当前版本面向“查看和给医生展示”，不再在网页里手动录入指标。后续复查后，把医院检查截图/PDF 和用药变化截图交给维护者整理，更新加密数据文件后网页会自动读取。

## 功能

- 医生摘要：基础情况、当前用药、最近异常与趋势重点
- 趋势图：血脂、肝功能、安全监测、甲功
- 指标表：全部结构化检查结果
- 时间线：复查、加药和方案调整
- 用药记录：当前及历史药物
- 打印和导出备份

## 隐私说明

GitHub Pages 网站本身是公开地址。结构化健康数据存放在 `data/records.enc.json`，使用浏览器本地解密，仓库内不保存查看密码，也不上传原始报告截图。

如果未来需要更新数据，需要用同一个查看密码重新生成加密数据文件。

## GitHub Pages

仓库设置中使用 `Settings` -> `Pages`，Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/root`。
